## Scripts

Clone repository:
```
git clone https://github.com/jsh15932/Python_Django_Social_Service.git
```

Install the requirements:
```
Django
```

Create the database:
```
python manage.py migrate
```

Run the development server:
```
python manage.py runserver
```

Server:
```
127.0.0.1:8000
```
